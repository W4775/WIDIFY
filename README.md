# :: Widify
<p>Widify is a chrome extension that allows you to utilize the full width of your wide or ultrawide monitor.</p>


Currently supports:

<ul>
  <li>Amazon.com</li>
  <li>Gamestop.com</li>
  <li>GitHub.com</li>
  <li>Reddit.com</li>
  <li>Slickdeals.net</li>
  <li>Walmart.com</li>
</ul>
